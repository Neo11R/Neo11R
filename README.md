### 📋 Características Principales



* **Gestión de Riesgo:** Implementación de Interés Compuesto (30x) y monitoreo de Drawdown en tiempo real.
* **Modo Hedge:** Capacidad para gestionar posiciones bidireccionales simultáneas.
* **Análisis Técnico:** Integración de indicadores dinámicos (RSI, MACD, Bollinger Bands, EMAs).
* **Alta Disponibilidad:** Sistema diseñado para ejecución desatendida 24/7 con logs detallados.
